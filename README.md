# 📱 Pokédex Flutter

Aplicación desarrollada en **Flutter** que consume la API pública **PokeAPI** para visualizar información de Pokémon mediante **Infinite Scroll** y búsqueda por nombre.

---

## 👩‍💻 Autor

**Alessia Pérez**

---

## 🎯 Objetivo

- Consumir una API REST utilizando Flutter.
- Implementar Infinite Scroll.
- Mostrar información detallada de Pokémon.
- Incorporar búsqueda por nombre.
- Aplicar diseño responsive para dispositivos móviles y web.

---

## ✨ Funcionalidades

✅ Infinite Scroll (5 Pokémon por carga)

✅ Búsqueda por nombre

✅ Información detallada de Pokémon

✅ Diseño responsive

✅ Consumo de API REST

✅ Compatible con Android y Web

---

## 🛠 Tecnologías Utilizadas

| Tecnología | Versión |
|------------|----------|
| Flutter | 3.44.1 |
| Dart | 3.12.1 |
| HTTP | 1.6.0 |
| Infinite Scroll Pagination | 5.1.1 |
| PokeAPI | API Externa |
